#📄 Currículo em HTML — Currículo Web Estruturado

👉 **Acesse aqui:**  
[Curriculo em HTML](https://baccarinha.github.io/curriculohtml/)

🎯 **Objetivo do Projeto**

Criar um currículo digital estruturado em HTML, permitindo apresentar informações profissionais de forma clara, organizada e acessível através da web.

🧩 **Funcionalidades do Sistema**

|O projeto apresenta as seguintes funcionalidades:|
|--|
📄 Exibição de currículo em formato digital
🧠 Organização das informações por seções
🔗 Inserção de links e elementos interativos
🖥️ Interface simples e objetiva
📱 Acesso direto pelo navegador
📋 Estrutura visual organizada
🎓 Projeto voltado ao aprendizado de HTML
🛠️ Tecnologias Utilizadas

|Este projeto foi desenvolvido utilizando:|
|--|
HTML5 — Estruturação do conteúdo
CSS3 — Estilização visual da página
GitHub Pages — Hospedagem do projeto
Inteligência Artificial (IA) — Apoio na criação e organização do layout
